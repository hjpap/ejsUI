EJS UI - Switch Button

以下是 Switch Button 的用法说明和实例：

Demo


State:

添加

1.引入JS和Css文件

<script src='ejs.core.js'></script>
<script src='ejs.promise.js'></script>
<script src='ui/eui.core.js'></script>
<script src='ui/eui.switchbutton.js'></script>
<link rel='stylesheet' type='text/css' href='css/eui.switchbutton.css'/>

2.require

//引用
var SwitchButton=require("EJS.UI.SwitchButton");
//初始化构造控件（hostElement为div节点）
var switchBtn=new SwitchButton(hostElement);
//显示
switchBtn.forceLayout();



属性和方法

1.doChange方法

switchBtn.doChange=function(){ alert(this.checked); }

doChange方法能够为switch Button添加事件，当改变状态时能够触发。

2.checked属性

switchBtn.checked//得到当前状态
switchBtn.checked=true//设置Switch Button状态为选择状态

checked属性可以得到Switch Button的当前状态；也可以设置Switch Button的状态。
